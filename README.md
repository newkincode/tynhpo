# 사용법
```
s : 스텍값
+ : 스텍에 1을 넣음
- : 스텍에 1을 뺌
? : if문 사용법 : 참일때 값 ? 조건 ? 거짓일때 값 이 리턴됨<
* : 문자열로 바꿈 예시 : +;+;s*; = 2에 해당되는 아스키코드값이 리턴됨
w : 반복 사용법 : w조건[코드]
^ : 스텍값에 {num}을 곱해서 그값을 저장 사용법 : ^{num}
! : 스텍값에 {num}을 나눠서 그값을 저장 사용법 : ^{num}
& : 스텍을 0으로 초기화
. : 출력 사용법 : .값
; : 한 코드가 끝나는걸 의미함
```
